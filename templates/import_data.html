File Format Instructions

Sales Import Format

Your Excel file should include the following columns:

  • Invoice Number - Unique identifier for the sale
  • Date - Date of the sale
  • Customer Name - Name of the customer
  • Product Name - Name of the product
  • Quantity - Number of units sold
  • Total Amount - Total amount including tax

Note: GST will be automatically calculated at 18%

Purchases Import Format

Your Excel file should include the following columns:

  • Order ID - Unique identifier for the purchase order
  • Date - Date of the purchase
  • Vendor Name - Name of the vendor
  • Product Name - Name of the product
  • Quantity - Number of units purchased
  • Total Amount - Total amount including tax

Note: GST will be automatically calculated at 18% and status will be set to "Delivered" by default
